# FINAL TAP

A fast, chaotic tap-survival browser game set in a collapsing “Doom Factory”.

## Play
**Live build (GitHub Pages):**
- https://farahoperates-tech.github.io/final-tap/

## How to run locally
1. Download/clone the repo
2. Open a terminal in the project folder
3. Run:
   - `python3 -m http.server 8000`
4. Open:
   - http://localhost:8000/

## Controls
- **Start / Initiate Containment** to begin
- **Pause / Resume**
- **Restart**
- **Spacebar** toggles pause (desktop)

## Notes
- Audio uses intro stings + looping factory ambience
- Includes “Audit Sweep” scavenger levels
